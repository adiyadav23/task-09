# 📊 Amazon Sales Dashboard – Data Analyst Task 8

## 📁 Project Overview
This project involves designing a **simple, interactive sales dashboard** using Tableau to visualize sales performance across **categories**, **regions**, and **time**. The dataset used is an Amazon sales dataset containing details such as order date, category, region, and sales.

---

## 🎯 Objective
- Create a clean and functional sales dashboard.
- Display key performance metrics using appropriate visualizations.
- Extract actionable business insights from the data.

---

## 🛠 Tools Used
- **Tableau** (for data visualization)
- **Amazon_Dataset.csv** (provided dataset)

---

## 📊 Dashboard Features
The dashboard includes the following charts:
1. **Line Chart** – Sales trends over time (Month-Year).
2. **Bar Chart** – Sales by Region.
3. **Donut Chart** – Sales by Category.
4. **Slicer** – Filter by Region or Category.

Color-coding was used to highlight top-performing regions and product categories.

---

## 🔍 Insights (Sample)
- The **Electronics** category consistently led in monthly sales.
- The **North** region had the highest total sales overall.
- A sharp increase in total sales was observed in **November**, likely due to seasonal promotions.
- The **Books** category showed fluctuating sales, possibly due to market seasonality.
